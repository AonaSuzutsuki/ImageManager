ClusterableFileStream Class

Namespace: ClusterableFileStream.IO


Overview


複数のファイルに跨いだファイル管理を提供します。読み書きするファイルは複数ですが、インタフェースを使う場合は仮想的な単一ファイルのように扱うことができるため、通常のFileStreamのように扱うことができます。ただし、Streamクラスは継承していないため多態性は保証しません。
FAT32などのファイルサイズに制限のあるファイルシステムでは大容量のファイル管理が行えませんが、書き込む内容によっては超えてしまう場合があります。そういった場面でこのクラスは力を発揮するでしょう。

Remark

ファイルの末尾に数字を追加する命名規則(file.dat.1, file.dat.2, ...)とするため、既にこれら一つでも同一名のファイルが存在する場合に正しく動作しない可能性があります。

Examples

GetSha256 サンプル

readonly byte[] testData = Encoding.UTF8.GetBytes("Hello,World!!");
const string expectedHash = "899948C9C72B65EBB59A8D01EE4DB68BABCF91D55C6D93DE66637EA869A3EF1D";

public void GetSha256Test1()
{
    var hash = "";
    using (var stream = new MemoryStream(testData))
    {
        hash = Sha256.GetSha256(stream);
    }
    Assert.AreEqual(expectedHash, hash);
}

public void GetSha256Test2()
{
    var hash = Sha256.GetSha256(testData);
    Assert.AreEqual(expectedHash, hash);
}

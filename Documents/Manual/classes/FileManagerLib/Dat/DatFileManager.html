ActionBytes(long, int, Action<byte[], int>, int)

public void ActionBytes(long start, int size, Action<byte[], int> action, int identifierLength = LEN);
パラメータ
long start データの開始位置を指定します。
int size 少しずつ読み込む際の毎回読み込む最大サイズを指定します。
Action<byte[], int> action 少しずつ読み込む際の実処理をデリゲートで指定します。デリゲートの第一引数は少しずつ読み込んだ際の実データで、第二引数は実際に読み込んだサイズが渡されます。
int identifierLength データの長さを格納するバイトサイズを指定します。 省略可
データを少しずつ読み込み、指定されたデリゲートの処理をデータ読み込み毎に実行します。

WriteToFile(long, string, string, int)

public bool WriteToFile(long start, string outFilePath, string expHash, int identifierLength = LEN);
パラメータ
long start データの開始位置を指定します。
string outFilePath 書き出すファイルパスを指定します。

Examples

GetSha256 サンプル

readonly byte[] testData = Encoding.UTF8.GetBytes("Hello,World!!");
const string expectedHash = "899948C9C72B65EBB59A8D01EE4DB68BABCF91D55C6D93DE66637EA869A3EF1D";

public void GetSha256Test1()
{
    var hash = "";
    using (var stream = new MemoryStream(testData))
    {
        hash = Sha256.GetSha256(stream);
    }
    Assert.AreEqual(expectedHash, hash);
}

public void GetSha256Test2()
{
    var hash = Sha256.GetSha256(testData);
    Assert.AreEqual(expectedHash, hash);
}
